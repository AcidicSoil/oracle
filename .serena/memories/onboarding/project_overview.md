Oracle is a TypeScript CLI wrapper around OpenAI Responses API and browser-driven ChatGPT automation. Main entrypoints are `oracle` and `oracle-mcp`. Repo areas include `src/` for CLI/browser/core logic, `tests/` for vitest coverage, `docs/` for user/developer docs, `scripts/` for support tooling, `config/` for MCP config, and `vendor/` for packaged notifier assets. The project uses Node 22+, pnpm, TypeScript, Vitest, oxlint, and oxfmt. See `AGENTS.md` for Oracle-specific browser/testing notes, including ChatGPT project URL handling and long-running Pro browser behavior.
